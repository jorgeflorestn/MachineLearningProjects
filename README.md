# Agregar entorno virtual a Jupyter Notebooks

1. Activar entorno virtual
2. `pip install ipykernel`
3. `python -m ipykernel install --user --name=myenv`
4. Seleccionar entorno kernel "myenv" dentro de Jupyter Notebooks
